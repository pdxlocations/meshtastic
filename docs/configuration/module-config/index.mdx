Modules are included in the firmware and allow users to extend the functionality of their mesh or device.

|                                    Name                                    |                                                                                                             Description                                                                                                             |
| :------------------------------------------------------------------------: | :---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------: |
|                <Link className={"button button--outline button--md cta--button"} to={"/docs/settings/moduleconfig/audio"}>Audio</Link>                |                                                                                      Enable Support for Codec2 Voice Comms on certain devices.                                                                                      |
|      <Link className={"button button--outline button--md cta--button"} to={"/docs/settings/moduleconfig/canned-message"}>Canned Messages</Link>       |                                                   Set a number of predefined messages to send out directly from the device with the use of an input device like a rotary encoder.                                                   |
|<Link className={"button button--outline button--md cta--button"} to={"/docs/settings/moduleconfig/external-notification"}>External Notification</Link>|                                                                Incoming messages are able to alert you using circuits you attach to the device (LEDs, Buzzers, etc).                                                                |
|               <Link className={"button button--outline button--md cta--button"} to={"/docs/settings/moduleconfig/mqtt"}>MQTT</Link>                   |                                              Forward packets along to an MQTT server. This allows users on the local mesh to communicate with users on another mesh over the internet.                                              |
|            <Link className={"button button--outline button--md cta--button"} to={"/docs/settings/moduleconfig/range-test"}>Range Test</Link>          | Send messages with GPS location at an interval to test the distance your devices can communicate. Requires (at least) one device set up as a sender and one as a receiver. The receiver(s) will log all incoming messages to a CSV. |
|            <Link className={"button button--outline button--md cta--button"} to={"/docs/settings/moduleconfig/serial"}>Serial Module</Link>           |                                                                                Send messages across the mesh by sending strings over a serial port.                                                                                 |
|  <Link className={"button button--outline button--md cta--button"} to={"/docs/settings/moduleconfig/store-and-forward-module"}>Store & Forward</Link> |                                                                         Stores messages on a device for delivery after disconnected clients rejoin the mesh.                                                                        |
|             <Link className={"button button--outline button--md cta--button"} to={"/docs/settings/moduleconfig/telemetry"}>Telemetry</Link>           |                                                                        Attach sensors to the device and transmit readings on a regular interval to the mesh.                                                                        |
|            <Link className={"button button--outline button--md cta--button"} to={"/docs/settings/moduleconfig/traceroute"}>Traceroute</Link>          |                                                                                Track which nodes are used to hop a message to a certain destination.                                                                                |
